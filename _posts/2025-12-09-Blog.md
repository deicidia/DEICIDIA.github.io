## Refonte du blog

Ce post sert de test dans l'utilisation de `Obsidian` pour l'écriture des billets de blog avec sauvegarde automatique sur `Github`. 

> block de test du rendu  
```c++
#include <iostream>
#include <vector>
#include <tuple>

auto main() -> int
{
    std::vector<std::tuple<int, int>> tuple_vec;
    tuple_vec.emplace_back(1, 2);

    for (auto [x, y] : tuple_vec)
        std::cout << x << " : " << y << " ";
    std::cout << "\n";
}
```
